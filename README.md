# Simulation d’un championnat de football à partir de modèles probabilistes

## Objectif

Ce projet propose de modéliser et de simuler une saison complète de Ligue 1 (2018–2019) à partir de données de matchs, en utilisant des outils probabilistes.

À partir de statistiques simples (buts marqués et encaissés), l’objectif est de reconstruire un classement réaliste des équipes et d’évaluer la pertinence de différents modèles.

---

## Approche

* estimation des performances offensives et défensives des équipes
* modélisation du nombre de buts par match (loi binomiale)
* simulation complète du championnat
* comparaison avec les résultats réels
* étude d’un modèle alternatif (loi de Poisson)

---

## Résultats

Le modèle proposé permet de reproduire de manière satisfaisante le classement de la saison, avec des écarts limités sur les points obtenus par les équipes.

Une comparaison avec un modèle de Poisson montre que, dans ce cadre, le modèle binomial fournit des résultats légèrement plus précis.

---

## Intérêt

Ce projet illustre comment des modèles probabilistes simples peuvent être utilisés pour analyser des données sportives et simuler des compétitions complètes.

Il met également en évidence l’importance du choix du modèle et des hypothèses sous-jacentes.
